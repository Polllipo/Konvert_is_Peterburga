# Конверт из Петербурга

Сайт производителя кастомной бумажной продукции для малого и среднего бизнеса: конверты, календари, гирлянды, подарочные сертификаты, печать логотипов.

**Live:** https://polllipo.github.io/Konvert_iz_Peterburga/

## Концепция

- Editorial / magazine-like эстетика: модульная сетка, контролируемая асимметрия, крупная display-типографика с кропом за край экрана
- Визуальная система построена на реальных фотографиях продукции, типографике и воздухе — без иллюстраций и 3D
- Моноширинная типографика как отсылка к печатной форме («printed but modern»)
- Палитра происходит из материалов: бумага, чернила, крафт

## Структура

```
index.html        — главная страница (single page)
konverty.jpg      — каталог: конверты
girlyandy.jpg     — каталог: гирлянды
kalendar.jpg      — каталог: календари
sert.jpg          — каталог: сертификаты
bumaga-hero.jpg   — hero: макро текстуры бумаги
```

## Технологии

- Чистый HTML + CSS, без сборки и зависимостей
- Шрифты: Martian Mono, Archivo (Google Fonts)
- Хостинг: GitHub Pages
